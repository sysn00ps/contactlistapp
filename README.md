<h3>Contact List App</h3>

<h5>Get the project</h5>
  git clone git@github.com:sysn00ps/contactlistapp.git

<h5>Install node modules</h5>
  npm install

<h5>Run mongoDB</h5>
  mongod

<h5>Run the app</h5>
  node server.js
